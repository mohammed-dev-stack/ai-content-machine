## 🎯 نظرة عامة

**AI Content Machine** هو تطبيق SaaS متكامل يولد محتوى جاهز للنشر على إنستغرام يشمل:

- ✨ **الهوك (Hook)** - سطر افتتاحي يجذب الانتباه (أقل من 12 كلمة)
- 📝 **الكابشن (Caption)** - وصف جذاب بطول 60-120 كلمة مع دعوة للتفاعل
- 🏷️ **الهاشتاغات (Hashtags)** - 7-10 هاشتاغ مدروس (واسع، متوسط، متخصص)
- 🖼️ **معاينة الصورة** - تمثيل بصري مع اسم المنتج والهوك

تم بناء المنصة خصيصاً للمطورين ومهندسي DevOps والمسوقين الرقميين الذين يحتاجون محتوى عالي التحويل في ثوانٍ.

## 🚀 التثبيت

### المتطلبات الأساسية
- Node.js 20+ 
- npm أو yarn
- مفتاح Groq API (مجاني من [console.groq.com](https://console.groq.com))

### الخطوة 1: استنساخ المستودع
```bash
git clone https://github.com/mohammed-dev-stack/ai-content-machine.git
cd ai-content-machine
```

### الخطوة 2: إعداد الواجهة الخلفية
```bash
cd backend
npm install
cp .env.example .env  # أضف مفتاح GROQ_API_KEY الخاص بك
npm run dev
```
الخادم يعمل على `http://localhost:3001`

### الخطوة 3: إعداد الواجهة الأمامية
```bash
cd frontend
npm install
npm run dev
```
التطبيق يعمل على `http://localhost:3000`

### الخطوة 4: اختبار الـ API
```bash
curl http://localhost:3001/api/test
# النتيجة المتوقعة: {"status":"ok","message":"AI route is working"}
```

## 📡 توثيق API

### `POST /api/generate`

توليد محتوى إنستغرام من تفاصيل المنتج.

**نص الطلب:**
```json
{
  "productName": "نظام مراقبة سيرفرات بالذكاء الاصطناعي",
  "audience": "مهندسو DevOps ومديرو النظم",
  "tone": "engineering"
}
```

**الرد (نجاح):**
```json
{
  "success": true,
  "hook": "تضيع ساعات في مراقبة سيرفراتك يدويًا؟ 🤖",
  "caption": "المشكلة: أنظمتك الحالية تتطلب مراقبة يدوية...",
  "hashtags": ["#DevOps", "#AI", "#ServerMonitoring", "#CloudComputing"]
}
```

**الرد (خطأ):**
```json
{
  "success": false,
  "error": "productName and audience are required"
}
```

### `GET /api/test`
نقطة نهاية للتحقق من الصحة.

**الرد:**
```json
{
  "status": "ok",
  "message": "AI route is working"
}
```

## 🌍 النشر

### الواجهة الخلفية (Render)
1. ارفع الكود إلى GitHub
2. أنشئ **Web Service** جديد على [render.com](https://render.com)
3. اربط المستودع
4. عيّن:
   - **Root Directory:** `backend`
   - **Build Command:** `npm install`
   - **Start Command:** `node server.js`
5. أضف متغير البيئة: `GROQ_API_KEY`
6. انشر

### الواجهة الأمامية (Vercel)
1. ارفع الكود إلى GitHub
2. استورد المشروع على [vercel.com](https://vercel.com)
3. عيّن:
   - **Root Directory:** `frontend`
   - **Framework:** Next.js
4. أضف متغير البيئة: `NEXT_PUBLIC_API_URL` (رابط Render الخاص بك)
5. انشر
